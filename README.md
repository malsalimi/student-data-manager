## ⚙️ الميزات الرئيسية

- تسجيل دخول آمن وإنشاء حساب جديد.
- إضافة وتعديل وحذف وعرض بيانات الطلاب.
- قائمة منظمة لعرض جميع الطلاب.
- تخزين محلي للبيانات.
- تصميم واجهات أنيق باستخدام Flutter.
